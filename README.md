# Hello World API

API simples feita em C# com ASP.NET Core que retorna "Hello World!" ao acessar a URL raiz.

## 🚀 Como rodar localmente

Pré-requisito: ter o [.NET SDK](https://dotnet.microsoft.com/download) instalado.

```bash
dotnet run
```

Depois abra o navegador em `http://localhost:5000` e vai aparecer `Hello World!`.

## 📦 Deploy

A API está hospedada no [Render](https://render.com) e o código está disponível neste repositório.

A cada `git push` na branch `main`, o Render atualiza automaticamente.

## 🛠️ Tecnologias

- C#
- ASP.NET Core (Minimal API)
- Docker
- Render (hospedagem)
